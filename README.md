# WAV Label Creator

**Author:** Xuedan Gao  
**Date:** 2026-01-05


A small GUI tool to embed WAV cue points and text labels into all `.wav` files in a selected folder.

## Requirements
- OS: Windows / macOS
- Python: 3.8+ (recommended 3.10+)
- Dependencies: standard library only

> This tool uses `tkinter` for the GUI (usually included with standard Python installs).

## Files
- `MerryChristmas.py`
- `config.json`

## How to Run
Open a terminal / command prompt in the script folder and run:
```bash
python MerryChristmas.py
